# プロジェクト概要

## 🌍 WorldSpeakAI とは

WorldSpeakAIは、最先端のAI技術を活用した多言語学習アプリケーションです。Gemini AIとの自然な音声会話を通じて、世界中の様々な言語でのコミュニケーション能力を身につけることができます。

## 🎯 ミッション

「世界中の人々が言語の壁を越えて、自由にコミュニケーションできる世界を実現する」

## 👥 ターゲットユーザー

### 主要ターゲット
- **多言語学習者**（初級〜上級）
  - 英語、中国語、スペイン語、フランス語、ドイツ語、韓国語など
- **ビジネスパーソン**（国際会議、海外出張、多国籍企業勤務）
- **専門職従事者**（医師、弁護士、エンジニア等の国際的な専門分野）
- **学術研究者・大学院生**（国際学会、多言語での研究発表）
- **学生**（留学準備、語学試験対策、第二外国語習得）
- **旅行者**（世界各国での会話練習）
- **移住・駐在予定者**（現地語習得、文化理解）

### 年齢層
- 15歳〜65歳（メインターゲット：20〜40歳）

### 学習言語（Gemini AI対応言語）
- **主要対応言語**：英語、中国語（簡体字・繁体字）、日本語、韓国語、スペイン語、フランス語、ドイツ語、イタリア語、ポルトガル語、ロシア語、アラビア語、ヒンディー語など
- **学習方向**：任意の言語ペア（例：日本語→英語、英語→中国語、スペイン語→フランス語など）

## ✨ 主要機能

### 1. AI多言語音声会話機能
- **リアルタイム音声認識**：Web Speech APIによる多言語音声認識
- **自然な会話生成**：Gemini AIによる各言語での文脈理解と応答
- **多言語音声合成**：expo-speechによる自然な各言語での音声出力

### 2. 革新的台本ガイド機能 🌟
- **初心者サポート**：各言語での会話開始時に台本を自動生成
- **言語ペア対応**：学習者の母語と学習言語の組み合わせに最適化
- **柔軟な対応**：台本から外れた発言にもAIが各言語で自然に対応
- **段階的成長**：学習進捗に応じて台本依存度を調整
- **文化的配慮**：各言語圏の文化的コンテキストを考慮した台本内容

### 3. 専門分野多言語対応機能 🌟
- **国際ビジネス**：各言語での業界別、職種別の専門用語と表現
- **学術分野**：多言語での研究発表、学会での議論、論文ディスカッション
- **技術分野**：IT、医療、法律、工学などの専門会話（各言語対応）
- **文化・社会**：各言語圏の文化的背景を考慮した会話設定
- **カスタムシナリオ**：ユーザーの専門分野と言語ペアに合わせた会話設定

### 4. ユーザー認証・プロフィールシステム
- **メール/パスワード認証**：Supabaseによる安全な認証
- **多言語プロフィール管理**：母語、学習言語、レベル、専門分野設定
- **学習履歴追跡**：各言語での会話履歴、進捗管理、専門用語習得状況

### 5. サブスクリプション機能
- **無料プラン**：月60分のマイク時間、基本台本機能、5言語対応
- **スタンダードプラン**：月15時間のマイク時間、全機能、全言語対応、履歴保存
- **プレミアムプラン**：月50時間のマイク時間、AIコーチング機能、詳細レポート、優先サポート

### 6. 多言語学習支援機能
- **マイク時間制限**：実際に話した時間のみカウントする公平な使用量管理
- **難易度調整**：各言語のユーザーレベルに応じた会話・台本調整
- **発音フィードバック**：言語別の発音、文法の改善提案
- **学習分析**：各言語での専門用語習得状況、会話スキル向上の可視化
- **AIコーチング機能**（Premium限定）：毎日の学習提案、詳細な進捗レポート、個別カリキュラム
- **言語間比較**：複数言語の学習進捗比較

## 🔧 技術的特徴

- **Web**：Chrome、Safari、Edge対応（初期リリース）

- **Android**：スマートフォン、タブレット対応（後期実装）

### AI技術の活用
- **Gemini AI**：100+ 言語での高精度な会話生成と専門分野対応
- **多言語台本生成AI**：言語ペアとシナリオに応じた最適な会話台本の自動生成
- **言語別学習最適化**：各言語の特性を考慮した個別カリキュラム調整

### リアルタイム多言語処理
- 低遅延の多言語音声認識
- 即座の各言語でのAI応答生成

### 1. 真のマルチリンガル対応
- **競合の課題**：多くのアプリは英語中心、または限定的な言語ペア
- **WorldSpeakAIの解決策**：Gemini AIの多言語能力を活用し、任意の言語ペアでの学習を実現

### 2. 台本機能の多言語革新性
- **競合の課題**：「何を話せばいいかわからない」問題が全言語で発生
- **WorldSpeakAIの解決策**：各言語の文化的コンテキストを考慮した台本自動生成

### 3. マイク時間制限システムの革新性
- **競合の課題**：複雑な制限体系（セッション数、文字数、トークン数など）でユーザー混乱
- **WorldSpeakAIの解決策**：「話した時間 = 使用時間」という直感的でフェアな制限方式

### 4. 専門分野での多言語対応
- **競合の課題**：日常会話中心で、専門的な多言語会話練習機能は限定的
- **WorldSpeakAIの解決策**：医療、法律、IT、学術など各分野の多言語専門用語・表現に対応

### 5. アグレッシブ価格戦略
- **競合の課題**：高価格で継続利用が困難（月額$20-50が相場）
- **WorldSpeakAIの解決策**：市場シェア獲得優先の破格価格設定（競合の半額以下）

### 6. 文化的配慮とローカライゼーション
- **実体験から生まれた解決策**：言語学習は単なる言葉の翻訳ではなく、文化的コンテキストの理解が重要という認識
